# ByteBites: A Campus Food-Ordering System

Built the core architecture for ByteBites, a campus food-ordering system, by using AI as a design partner to generate and refine UML diagrams and Python scaffolds, then applying object-oriented design and simple algorithms to implement reliable menu, ordering, and recommendation logic.

- [x] Used object-oriented design to break a feature request into core components.
- [x] Represented and manipulated structured data using simple Python classes and containers.
- [x] Applied algorithmic reasoning to tasks such as sorting, filtering, and validation.
- [x] Used UML-style diagrams to think about system structure before writing code.
- [x] Incorporated AI-produced code and diagrams as inputs to the workflow.
